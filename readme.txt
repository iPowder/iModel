Installation Process:
1. Download the LabVIEW 2023 Q1 Runtime installer from the labview_runtime folder or the link https://www.ni.com/en/support/downloads/software-products/download.labview-runtime.html#477317 and proceed with the installation to any location.

Alternatively, download all files from the runtime_installer folder on this page, unzip the runtime_installer.zip.001 file to any location (the unzip program will automatically find the 002 file and unzip together). After extraction, run runtime_installer.exe to install LabVIEW 2023 Q1 Runtime. Once installed, you can delete the both runtime_installer zip file and all its contents.


2. Download the iModel.zip.001 file from the iModel_program folder and unzip it to any location you prefer. If you have completed the LabVIEW 2023 Q1 Runtime installation, click on iModel.exe within the extracted iModel folder to run the program.

Please note that iModel.exe requires LabVIEW 2023 Q1 Runtime to run.

Requires Windows 10 and above systems.
